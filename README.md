## Getting Started

Steps to use this repository locally:

- Install packages

```bash
npm install
# or
yarn install
```

- Run dev build

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
